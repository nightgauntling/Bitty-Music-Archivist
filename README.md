# Bitty-Music-Archivist
Bitty Music Archivist is a personal side project focused on improving my Python skills by creating a lightweight music management tool for organizing and renaming my local library. Built for personal use, with potential for future expansion and public release as a themed archive tool.

---

## Features (Current & Planned)

### Renaming
- Replace underscores with spaces
- Rename files using metadata (e.g. `Title - Artist.mp3`)
- Future support for customizable naming rules

### Organization
- Sort files into artist or band folders
- Future support for genre-based and advanced organization

### Safety
- Dry-run preview before applying changes
- User confirmation before execution
- Operation history for undo functionality
- Optional backups before modification

---

## Goals

- Provide a safe and simple workflow for cleaning up music libraries
- Build a flexible system for metadata-based organization
- Develop a foundation for a more advanced archival tool
